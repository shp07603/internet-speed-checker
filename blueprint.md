# My Network Score - Project Blueprint

## 프로젝트 개요
사용자의 인터넷 속도(다운로드, 업로드)와 품질(핑, 지터, 패킷 손실)을 종합적으로 측정하고 분석하여 네트워크 점수를 매겨주는 웹 애플리케이션입니다.

## 기능 및 디자인 상세
- **종합 분석:** 다운로드/업로드 속도뿐만 아니라 반응속도, 연결 떨림, 데이터 씹힘(패킷 손실)을 측정합니다.
- **실시간 게이지:** 측정 중인 속도를 시각적으로 보여주는 SVG 게이지 UI.
- **상태 카드:** 각 측정 항목별 상세 결과를 카드 형태로 표시 (상태에 따라 색상 변경).
- **실시간 차트:** 최근 30회의 반응속도 히스토리를 바 차트로 시각화.
- **최종 등급 및 조언:** 측정 결과를 바탕으로 A~F 등급을 부여하고, 개선을 위한 팁을 제공합니다.
- **디자인 컨셉:** 'Space Mono' 폰트와 어두운 배경, 형광색 포인트(네온 블루, 그린, 오렌지)를 사용한 사이버네틱하고 전문적인 느낌의 UI.

## 현재 구현 상태
- `index.html`: 구조 정의 및 외부 폰트/스타일/스크립트 연결.
- `style.css`: 현대적인 CSS 기능(Variable, Flexbox, Grid, Animation)을 사용한 전체 스타일링.
- `main.js`: 
  - `fetch` API(no-cors 모드 포함)를 이용한 핑 및 속도 측정 로직.
  - 가상 데이터가 아닌 실제 네트워크 요청을 통한 측정 시도.
  - 결과 분석 및 UI 업데이트 로직.

## 향후 계획 (제안)
1. **정확도 향상:** 더 많은 측정 서버 확보 및 알고리즘 고도화.
2. **히스토리 저장:** 로컬 스토리지(LocalStorage)를 사용하여 이전 측정 기록 저장 및 비교 기능.
3. **공유 기능:** 결과를 이미지로 저장하거나 링크로 공유하는 기능.
4. **상세 분석 리포트:** 네트워크 환경에 대한 더 깊은 기술적 분석 데이터 제공.
